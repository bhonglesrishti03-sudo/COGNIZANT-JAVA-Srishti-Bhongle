# CSS3 Module — Local Community Event Portal

## Cognizant 3-Week Upskilling Program

This folder contains the CSS3 module of the Local Community Event Portal.
All styling is in a single external `styles.css` file linked to `index.html`.

---

## File Structure

```
CSS3/
├── index.html      → Full portal page (all CSS exercises visible here)
├── styles.css      → All CSS — organised by exercise with comments
├── help.html       → Help page (shares styles.css)
└── images/         → Place events1.jpg to events6.jpg here
```

---

## How to Run

1. Open `index.html` in Chrome
2. Open DevTools (F12) to inspect and test styles live
3. Use Chrome's device toolbar to test responsive breakpoints
