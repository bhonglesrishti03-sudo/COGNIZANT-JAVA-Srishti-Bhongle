Frequently Asked Questions

Q: Why does the location feature ask for permission?

The Geolocation API requires browser permission to access your device location. You can allow or deny this in the popup that appears.

Q: Will my preferences be saved if I clear my browser data?

No. Preferences are stored in localStorage which is cleared when you clear browser site data.

Tip: Use Chrome DevTools (F12) to inspect any page, view console logs, and debug JavaScript.
