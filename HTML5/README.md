# HTML5 Module — Local Community Event Portal

## Cognizant 3-Week Upskilling Program

This folder contains all 10 HTML5 exercises for the Local Community Event Portal project.

---

## File Structure

```
HTML5/
├── index.html                  → Portal home / navigation hub
├── ex1_baseTemplate.html       → Exercise 1  : HTML5 base document structure
├── ex2_navigation.html         → Exercise 2  : Anchor-based navigation & sections
├── ex3_welcomePage.html        → Exercise 3  : Welcome banner, ID/class selectors
├── ex4_imageGallery.html       → Exercise 4  : Image gallery using table layout
├── ex5_registrationForm.html   → Exercise 5  : Event registration form
├── ex6_eventFeedback.html      → Exercise 6  : Event handling (blur/change/click/dblclick/keyup)
├── ex7_videoInvite.html        → Exercise 7  : Video element & media events
├── ex8_userPreferences.html    → Exercise 8  : localStorage & sessionStorage
├── ex9_geolocation.html        → Exercise 9  : Geolocation API
├── ex10_devtools.html          → Exercise 10 : Chrome DevTools debugging
├── help.html                   → Help page (opens in new tab)
└── images/                     → Place event1.jpg to event6.jpg here
```

---

## How to Run

1. Clone or download this repository
2. Open `index.html` in Google Chrome
3. Navigate to any exercise from the portal home page

> No server required — all files are plain HTML/CSS/JS running in the browser.

| 9 | ex9_geolocation.html | getCurrentPosition, enableHighAccuracy, timeout, error codes |
